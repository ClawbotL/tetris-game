# Tetris Game

A fully-featured Tetris game built with vanilla HTML, CSS, and JavaScript.

## Features

- Classic Tetris gameplay
- Score tracking
- Next piece preview
- Hold piece
- Levels
- Local leaderboard
- Responsive design
- Dark/light mode toggle
- Keyboard controls
- Game over screen
- Pause functionality

## Controls

| Key       | Action               |
|-----------|----------------------|
| ←         | Move left            |
| →         | Move right           |
| ↓         | Soft drop            |
| ↑         | Rotate               |
| Space     | Hard drop            |
| C         | Hold piece           |
| P         | Pause/Resume         |

## How to Play

1. Open `index.html` in a modern web browser
2. Use the controls listed above
3. Clear lines to earn points and level up
4. Try to beat your high score!

## Development

This game uses vanilla JavaScript, HTML, and CSS - no dependencies.
